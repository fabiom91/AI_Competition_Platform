Disclaimer

In order to make a Machine Learning Competition submission, you must provide the followings: a Results Dataset which contains the competition test data labelled using your model; a Link to your Code that can be a GitHub repository, a Google Drive, Dropbox etc. and should contain you model, eventual resources and a readme with the model description and usage instructions.

Please review the following requirements before submitting your results:

  • The submission does NOT contain null values;
  • The predicted feature is named: 'class' (lowercase) regardless of the type of model you've built;
  • If your submission does not follow the order of the validation set, it has to match its index;
  • The evaluation measures are in their normal range
  • The predicted class data-type match the class of the Training/Validation dataset.

Please, be sure to check out the article How to join a Competition for a detailed description of the above requirements.

Fail to provide the required components could results in a automatic withdrawal from the competition.
